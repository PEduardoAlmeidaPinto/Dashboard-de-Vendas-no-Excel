# Dashboard-de-Vendas-no-Excel
Projeto de Dashboard de Vendas da XBOX com foco em análise exploratória e visualização de dados. Apresenta KPIs como faturamento, volume de vendas, desempenho por período e comparação de resultados, facilitando insights estratégicos por meio de uma interface clara e interativa.

## Base de Dados
A base de dados utilizada neste projeto é fictícia e foi disponibilizada para fins educacionais, permitindo que outros usuários pratiquem análises e criem suas próprias visualizações.
